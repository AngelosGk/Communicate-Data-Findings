Introduction¶

This project explore the mobile Apps that users prefer. This dataset is consisted by variables that help us to reach in plenty of conclusions about what make users spend more money. This dataset being obtaining from free Kaggle datasets called Mobile App Store in CSV format file. Below we can see the exact meaning of column names.

1.  "id" : App ID
2.  "track_name": App Name
3.  "size_bytes": Size (in Bytes)
4.  "currency": Currency Type
5.  "price": Price amount
6.  "user_rating" : Average User Rating value (for all version)
7.  "userratingver": Average User Rating value (for current version)
8.  "prime_genre": Primary Genre
9.  "sup_devices.num": Number of supporting devices
10. "lang.num": Number of supported languages
